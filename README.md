# SIMULAÇÃO CLÍNICA ABERTA

Plataforma de simulação clínica aberta desenvolvida em Next.js, dedicada ao aprimoramento de habilidades em enfermagem através de cenários realistas de simulação.

## Sobre o Projeto

Este projeto é uma aplicação web que fornece uma plataforma colaborativa para:
- Explorar cenários de simulação clínica interativa
- Aprender através de casos práticos com feedback imediato

## Tecnologias Utilizadas

- **Next.js 16.2.1** - Framework React com suporte a Server Components
- **React 19.2.4** - Biblioteca UI
- **TypeScript 5** - Tipagem estática
- **Tailwind CSS 4** - Framework de CSS utilitário
- **Lucide React 1.7.0** - Biblioteca de ícones
- **ESLint 9** - Linting de código

## Funcionalidades Implementadas

### Página Inicial (Home)
- Header com navegação
- Seção de boas-vindas com descrição da plataforma
- Links de navegação (Início, Sobre, Desenvolvedor)
- Botões de ação para explorar simulações

### Listagem de Simulações
- Exibição de todos os cenários disponíveis
- **Filtro por Categorias (Tags)**: Permite filtrar cenários por categoria específica
- Contagem de cenários por categoria
- Botão flutuante para adicionar novos cenários
- Modal para criar novo cenário
- Responsividade mobile-first
